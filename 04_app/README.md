# How to use the App

![Image 1](01_images/utils/app1.PNG)

The uploaded file *MUST* contain the following columns:

* Income
* NumCatalogPurchases
* NumWebPurchases
* MntFruits
* NumStorePurchases
* MntMeatProducts
* MntSweetProducts
* MntWines
* MntFishProducts
* Age
* MntGoldProds
* Recency
